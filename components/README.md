# Components

Three representative React components from the production frontend, sanitized for public review.

| Component | What it shows |
|---|---|
| `AlphaScanFeed` | Realtime subscription pattern, list virtualization, chain switching |
| `SafetyPanel` | Pre-trade gating, AI fallback handling, cached scan UX |
| `QuickSwap` | Trade execution flow shape, quote refresh, error handling |

## Withheld

- Privy embedded wallet integration (`useWallet`, password unlock UI)
- Trade execution internals (signing, RPC submission, retry logic)
- Bridge panel (Li.Fi quote and route selection)
- Trigger order panel (Jupiter triggers integration)
- Admin pages and treasury views
- shadcn/ui base setup and design tokens
